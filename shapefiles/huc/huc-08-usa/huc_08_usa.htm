Description:
Abstract:
8-digit hydrologic unit codes, derived from the USGS 1:2,000,000 Hydrologic Units (huc2m) layer. The original USGS layer and metadata was downloaded from: <http://nationalatlas.gov/hucsm.html>. The oringal 8-digit HUC layer from USGS was reprojected in UTM NAD 1983 Zone 15N to create this layer.

This digital, geographically referenced data set was developed by the Iowa Department of Natural Resources to carry out agency responsibilities related to management, protection, and development of Iowa's natural resources.

Attribute:
Attribute_Label: HUC
Attribute_Definition:
The Hydrologic Unit Code. The code is 8 digits and can be subdivided as follows: >Digits Type >1-2 Water Resource Region >1-4 Subregion >1-6 Accounting Unit >1-8 Cataloging Unit > This is the same as Cat. There are six polygons in the file with a Huc of 0; these polygons are not assigned to any hydrologic unit.
